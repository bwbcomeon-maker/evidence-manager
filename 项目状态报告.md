## 🎯 下一步操作

### 步骤 1: 安装 Node.js（必需）

1. **下载 Node.js**
   - 访问：https://nodejs.org/
   - 下载 LTS 版本（推荐 18.x 或更高）
   - Windows 系统下载 `.msi` 安装包

2. **安装 Node.js**
   - 双击安装包
   - **重要：** 安装时务必勾选 "Add to PATH"
   - 完成安装

3. **验证安装**
   - 重新打开 PowerShell
   - 运行：
     ```powershell
     node --version
     npm --version
     ```
   - 应该显示版本号

### 步骤 2: 启动前端项目

**方式一：使用启动脚本（推荐）**
```powershell
cd d:\evidence-manager\frontend
.\start.ps1
```

**方式二：手动启动**
```powershell
cd d:\evidence-manager\frontend
npm install
npm run dev
```

### 步骤 3: 访问应用

- **前端：** http://localhost:3000
- **后端：** http://localhost:8081
- **后端健康检查：** http://localhost:8081/actuator/health
